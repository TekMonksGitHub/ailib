Module: entropy - Stream entropy generator.

Methods

(static) getEntropy(stream, scratchpad) → {object}

Stream entropy generator. The stream can be added to, if the scratchpad is sent back from the previous calculation, as an input to the new stream being added. The stream is an array of symbols.

scratchpad.entropy is the resulting entropy. Worst case - 2*O(n) where n is the stream length so far.

Parameters:
Name Type Description
stream array An array of symbols whose entropy needs to be calculated.
scratchpad object The scratchpad returned from calculation.
